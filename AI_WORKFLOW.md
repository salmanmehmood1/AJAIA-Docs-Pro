# AI Workflow Note

## AI tools used

- GitHub Copilot for scaffolding, iteration speed, and UI/API boilerplate acceleration

## Where AI sped things up

- Initial project structure and repetitive CRUD/UI wiring
- Drafting the README and architecture documentation
- Fast refinement of styling and component organization

## What was changed or rejected

- Reduced overbuilt solutions in favor of a simpler JSON-persistence design
- Kept file import intentionally limited to `.txt` / `.md` rather than accepting unsupported formats
- Reviewed and simplified generated UI code to keep the app readable and maintainable

## Verification approach

- Manually checked the frontend/backend flow design
- Added an automated backend test for document creation and sharing
- Included local setup instructions so the full flow can be reproduced easily
